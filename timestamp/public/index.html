Neeilan's Timestamp Microservice


How it works

You can pass a string as a parameter, and the app will check to see whether that string contains either a unix timestamp (example: 1451606400) or a natural language date (example: January 1, 2016), and you will be returned the date in both UNIX timestamp and natural formats.


You can also query the endpoint at neeiltimestamp.herokuapp.com/api using the following syntax:
neeiltimestamp.herokuapp.com/api?date=[TIMESTAMP] or neeiltimestamp.herokuapp.com/api?date=[URI_ENCODED_STRING]. For example, using the above text input field for December 15th, 2015 is equivalent to the following request:neeiltimestamp.herokuapp.com/api?date=December%2015,%202015
